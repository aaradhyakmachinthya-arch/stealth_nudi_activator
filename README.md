# stealth_nudi_activator

**Project Description:** A navigator that bypasses UI interaction, but still opens the file location. Use with caution. Close the file location to get Nudi Kannada on your device, or any other language if you decide to choose it from the file location. If the file is closed, your backup language translator opens, so make sure Kannada is your backup language.

## Stealth Nudi Kannada Activator ??
An elegant Python automation script that bypasses the multi-step frontend UI of Multilingual Nudi 6.5 to silently activate the background Kannada typing engine.

## How It Works ???
* Launches a clean native `tkinter` file browser.
* Triggers the 32-bit Nudi launcher via an absolute system path execution hook.
* Exploits the software's native cancel-fallback routine: when the explorer window is dismissed, the script forces the core Kannada engine to load straight into the Windows system tray without opening the main language menu.

## Features ?
* **Zero UI Distraction**: No manual language selection or menu clicking required.
* **Stealth Background Initialization**: Drops the keyboard translation engine directly into active memory.
* **Cross-Architecture Awareness**: Successfully navigates 32-bit application execution within a native 64-bit Windows environment.

## ?? Use with Caution!!
> **Important Note:** Forcefully close it in Task Manager if you need to return to your global English typing layout.
